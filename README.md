# 3D Web Project

A web project featuring 3D integration and interactive components.

cute-squirrel-1a3bcd.netlify.app

## Project Structure

- `index.html` - Main HTML file
- `script.js` - JavaScript functionality
- `style.css` - Styling
- `components/` - React components
  - `demo.tsx` - Demo component
  - `ui/` - UI components
    - `stagger-testimonials.tsx` - Staggered testimonials component
- `test_spline/` - Spline testing directory
- `.agent/` - AI agent skills for Spline 3D integration

## Features

- 3D web integration
- Interactive components
- Responsive design

## Getting Started

1. Open `index.html` in your web browser
2. Customize the styling in `style.css`
3. Modify functionality in `script.js`

## Technologies

- HTML5
- CSS3
- JavaScript
- React/TypeScript (for components)
- Spline 3D Integration
